1. Write a program named counter.cpp which increments then prints a counter every second. The program should run until the user presses enter.

    Hints:
    • Make a thread which continually increments, prints, then sleeps.
    • Use a global boolean variable to control the thread's loop.
    • Use std::getline (or std::ignore) on the main thread to wait for the user to press enter.
    • Set the global variable to false, then just wait for the thread to exit.
  2. Write a program named letters.cpp which runs for 20 seconds. The program should consist of three threads.
    • The first thread should print the letter "A" every second
    • The second thread should prints the letter "B" every 2 seconds
    • The third thread should print the letter "C" every 5 seconds
    You must use the same function for all three threads; use parameters to alter each thread’s behavior.

    Hints:

    • You do not have to check the actual time—e.g., you may assume that if you sleep 10 times for 2 seconds each time then it has been 20 seconds
    • Use std::unitbuf to turn off buffering for std::cout
